The Class & Class Declaration


The Class

You declare a class by specifying the class keyword followed by a non-reserved identifier that names it. A pair of matching open and close brace characters ({ and }) follow and delimit the class's body. This syntax appears below:

Class Declaration

// Declare using
// 'class' keyword
class identifier
{
   // class body
}
  • By convention, the first letter of a class's name is uppercased and subsequent characters are lowercased (for example, Employee).
  • If a name consists of multiple words, the first letter of each word is uppercased (such as SavingsAccount). This naming convention is called CamelCasting.

The following example declares a class named Book:

class Book
{
   // class body
}

Now, What inside a Class? A class's body is populated with fields, methods, and constructors. Combining these language features into classes is known as Encapsulation.

This capability lets us program at a higher level of abstraction (classes and objects) rather than focusing separately on data structures and functionality.
