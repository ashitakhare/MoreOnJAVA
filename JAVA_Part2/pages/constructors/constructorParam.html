Parameterized Constructor


First of all, Before Starting Did you come across a Question?? that What is the purpose of constructor?

As we all know there are Data members like : studentName, studentAge, studentDOB etc. What if we print them without assigning them a value.

Confused?????

Let us go through an Example.

Imagine the Student has a name as an instance variable, and you want the programmer using your Student class to set the name of the new Student. How could you do it? 

						
public class Student {
	String name; 
	
}

So, What do you think the value of name will be? Since we have not set any value, Compiler sets the default value to the name

For String Default value is null

What if instead of these default values I want to set my own Values as default. For Examples: as null does not sound good, I want to set Ashita as my Default value

How to do it?? Yes, Through Parameterized Constructors

The best place to put initialization code is in the constructor. And all you need to do is make a constructor with arguments.

Let us see how this can be done.

public class Student{
	String name;
	
	public Student (String newName){
		name = newName; // name assigned to value "Ashita"
	}
}

public class UseStudent{
	public Static void main(String args[]){
		
		Student student = new Student("Ashita"); // Parameterized Constructor
	}
}
